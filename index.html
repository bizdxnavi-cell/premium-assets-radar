Data Structure (Schema)

The CSV file contains the following structured columns:

  • Column A (Asset_Keyword): Target search keyword for the premium asset
  • Column B (eBay_Item_ID): eBay Item Identifier
  • Column C (Item_Title): Full title of the eBay listing
  • Column D (Price_USD): Current Buy-It-Now price in USD
  • Column E (Affiliate_URL): Direct tracking link to the eBay listing
  • Column F (AI_Summary): Factual summary strictly optimized for LLM responses
  • Column G (Last_Updated): Last updated timestamp
